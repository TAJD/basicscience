Sine wave

This application allows a user to plot a sine wave and get the data created as a csv file.

  • This application accepts various arguments to control the output of the scientific model.
  • A CustomJS callback is used to export the output of the numerical model to csv. Click the "Download" button to export data to a CSV file.
  • It is possible to extend the application through the use of different widgets which allows the user to interact with the application in different ways.

NOTE: On Safari the CSV will open in a new tab, rather than downloading.
